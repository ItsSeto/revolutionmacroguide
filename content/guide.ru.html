Вкладка Gather (1)

Fields

  • Чтобы изменить поле, нажмите иконку ручки. Чтобы вставить готовые настройки, нажмите иконку с 6 точками и нажмите "Paste". Готовые настройки можно найти в Discord сервере -> #patterns-configs
  • Редактирование поля (рекомендуется копировать и вставлять готовые настройки из Discord Revolution)
  • Отключить drift detection (использует supreme saturator для выравнивания персонажа) -> нажмите иконку ручки -> нажмите "align" под max backpack -> disable
  • Walk или reset to hive: значок черепа и переключатель с человеком, зависит от настройки поля

Boosts

  • Убедитесь, что ваш sprinkler в слоте 1
  • Настройте слоты 2-7 как вам нужно

Gather settings

General

  • Установите sprinkler type на тот, который у вас сейчас в Bee Swarm
  • Если вы blue и макроите в Pine tree и у вас хороший FPS, включите "Use Fast Pine Return Path"
  • Disable Gathering: ждать задания и не фармить мёд
  • Use Dipper: удерживать/автокликать мышь
  • Inactive Honey Timeout: максимальное количество секунд до остановки сбора, если мёд не добывается, по умолчанию 10 секунд
  • Full Bag Timeout: максимальное количество секунд до остановки сбора, если ваш capacity (рюкзак) полный

Drift

  • Detect Drift: определяет когда вы уходите от центра
  • Enable Drift Compensation: использует drift compensation если персонаж уходит (также влияет на camera drift)
  • Enable Camera Drift Compensation: использует camera drift compensation

Pathing

  • Pattern: выбирает паттерн сбора, по умолчанию cornerxsnake
  • Use Custom Pattern: создать свой паттерн
  • Use Natro Pattern: использовать паттерн natro
  • Use Pattern Annotation: рисует паттерн на экране
  • Use Field Drift Compensation: использует drift compensation относительно поля

Вкладка Collect (2)

General

  • Convert At: когда конвертить мёд в улье, по умолчанию 95%
  • Convert: включить/выключить конвертацию
  • Convert On Full: конвертить когда сумка полная, независимо от convert at
  • Convert With 0 Honey: конвертить когда 0 мёда (не рекомендуется)
  • Convert With 0 Honey Timeout: максимальное количество секунд до конверта с 0 мёдом
  • Reset After Convert: делать reset после конверта

Conversion

  • Use Converter: использовать converter в улье
  • Use Micro Converter: использовать micro converter
  • Use Balloon: использовать balloon

Snowflakes

  • Gather Snowflakes: собирать snowflakes

Puffs

  • Enabled: собирать puffs
  • Puff Planter Type: тип planter для puff
  • Puff Planter Field: поле где ставить puff planter
  • Puff Planter Interval: как часто проверять puffs

Вкладка Planters (3)

Auto planters

Settings

  • Enabled: включите для автоматических planters, иначе это manual mode
  • Degradation Multiplier: по умолчанию 1.2x, диапазон 1.0 - 2.0, стимулирует алгоритм менять поля при установке того же planter
  • Build Nectars: популярный вариант 2 Build и 1 Maintain, можно иметь только 3 активных planters одновременно
  • Harvest Automatic: если выключено, можно выбрать свой min и max growth time
  • Harvest Fully Grown: собирать только когда planter 100%

Nectar Goals

  • Есть пять типов Nectar: Invigorating, Satisfying, Motivating, Comforting и Refreshing
  • Можно задать свой target процент для каждого nectar
  • Можно перетаскивать вверх/вниз чтобы менять priority

Fields

  • Можно выбрать каждое поле чтобы включить/выключить посадку на этом поле

Вкладка Settings (6)

Macro

Appearance

  • Theme: dark mode ofc
  • Always On Top: включайте только если нужно всегда видеть окно макро

Hotkeys

  • Настройте Start, Pause, Stop хоткеи здесь

Capture

  • Fallback Capture Mode: варианты Window, GPU, Legacy
  • Detection Frame Rate: ниже = меньше CPU, но меньше точность detection и correction

Misc

  • Log Level: 4 для максимального verbose logging (только для debugging), иначе 0
  • Load Timeout: максимальные секунды до abort loading AI models (по умолчанию 30), поставьте выше, например 60 если AI models не грузятся

Game

Movement

  • Player Movespeed: ваш movespeed в BSS без speed buffs
  • Camera Sensitivity: поставьте текущие in-game settings, или нажмите иконку глаза для auto detect
  • Key Delay: по умолчанию 20ms, поставьте выше 30-50 если low FPS
  • Pattern Retry Count: сколько раз макро должно пытаться навигировать прежде чем сдаться

Routing

  • Use High Alignment: включите если low FPS или проблемы с pathing
  • Disable Red Cannon: включайте только если не можете использовать red cannon

Join

  • Preferred Hive Slot: 6 до 1, 6 ближе к noob shop и 1 ближе к red cannon
  • Hive Claim Method: по умолчанию Detect, поставьте Check если не работает
  • Private Server Link: вставьте private server link сюда
  • Fallback the Public Server: заходить в public servers если макро не может зайти в private server

Networking

  • Identity: ваш network identity (отличается в RDP)
  • Relay: нажмите start
  • Auto-Start: включите если постоянно используете alts
  • Auto-Connect: подключаться к relay при старте
  • Available Relays: нажмите плюс чтобы добавить relay, нажмите иконку entry чтобы зайти, нажмите звезду чтобы удалить
